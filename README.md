# Just-4-us

![Project Preview](/preview.png)

A small Valentine’s Day front-end project built using HTML, CSS, and vanilla JavaScript.  
Created as a vibe-coded, personal project focused on simplicity and interaction.

## About

This project is a lightweight web experience made for Valentine’s Day.  
It avoids frameworks and external libraries, keeping everything simple and readable.

## Tech Stack

- HTML  
- CSS  
- Vanilla JavaScript
